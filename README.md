# Installing Mobius

```bash
npm install --save @etchteam/mobius
```

# Usage

```jsx
import React from 'react';
import { Button } from 'mobius';

const MyComponent = () => (
  <Button
    type="primary"
    href="https://reactjs.org"
    target="_blank"
    rel="noopener noreferrer"
  >
    Learn React
  </Button>
)
```
